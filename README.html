Projeto de Automação de Relatório de Vendas

Descrição

Este projeto visa desenvolver uma automação de processos de negócios focada na geração de relatórios de vendas. A automação coleta dados de vendas, realiza cálculos e gera relatórios que podem ser utilizados para análise de desempenho de vendas.

Estrutura do Projeto

1. Definição do Fluxo do Projeto

a. Coleta de Dados

  • Fonte de Dados: Uma planilha Excel contendo informações sobre as vendas.

2. Planilha de Vendas

A planilha Excel criada inclui os seguintes dados:

  • Data da Venda
  • Produto
  • Quantidade Vendida
  • Preço Unitário
  • Total da Venda
